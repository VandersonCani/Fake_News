# Fake\_News





O objetivo do projeto é atingir jovens e adultos com informações verdadeiras e conscientização sobre a possibilidade de encontrar conteúdo com desinformação durante a navegação, aliás a desinformação tem um poder de crescimento e divulgação mais facilitados por ser fantasioso, tendencioso, de fácil entendimento lógico por não se basear em fatos, o que facilita que as pessoas guardem as informações falsas com mais facilidade e passem adiante, enquanto que informações verdadeiras com base teórica e prática, cientifica e histórica, tendem a desaparecer em meio a falsas informações por serem difíceis de absorver e repassar adiante.





Neste projeto utilizamos puramente HTML e CSS, a principio esteve sendo executado apenas com o live server por atender aos requisitos de testes do que foi implantando no projeto web até o momento.



Pasta Base\_Projeto - Arquivos de fotos com Requisitos Solicitados pelo cliente, que devem estar presentes no projeto.



Pasta Css - contém arquivos de CSS separados especificamente para facilitar a manutenibilidade.



Pasta js - Contem arquivos Javascript.



Pasta Pages - Contem os arquivos HTML das Páginas.



Pasta Componentes - Contém Footer e Header que são carregados separadamente por serem iguais para todas as páginas.
